# 🧠 Planora

<br>

Planora is a modern web application designed to help you plan, track, and analyze your personal development.  
Its goal is to make your objectives clearer and your daily/weekly progress more sustainable.

---


## 🚀 Features

<br>


- 🎯 Goal setting and tracking  
- 🗓️ Daily / weekly planning  
- 📊 Progress analytics and reporting  
- ⏱️ Habit tracking  
- 🧩 Simple and user-friendly interface

## ⚙️ Installation
<br>


### 1. Clone the repository

```bash
git clone https://github.com/omerdemirell/planora.git
cd planora
```
<br>

### 2. Run the Backend

```bash
cd backend
dotnet run
```
<br>

### 3. Run the Frontend

```bash
cd frontend
npm install
ng serve -o
```
